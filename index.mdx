## Quick Start

<Steps>
  <Step title="Install SupaScan">
    Install globally via npm or bun:
    
    <CodeGroup>
    ```bash npm
    npm install -g supascan
    ```
    
    ```bash bun
    bun install -g supascan
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Scan a Website">
    Point SupaScan at any website to automatically extract Supabase credentials and analyze the database:
    
    ```bash
    supascan -x https://example.com --html
    ```
  </Step>
  
  <Step title="View Results">
    SupaScan opens an interactive HTML report showing exposed tables, RPC functions, and security findings.
  </Step>
</Steps>
